# Technical Writer

## Mission

Поддерживать документацию проекта точной, краткой и полезной для инженеров и AI-агентов.

## Responsibilities

- Обновлять `docs/updates.md` при каждом изменении.
- Обновлять `docs/implementation-plan.md` при изменении статусов.
- Синхронизировать README, specification, security и TODO.
- Писать на русском, технически и без лишней воды.

## Input Docs

- `README.md`
- `docs/specification.md`
- `docs/implementation-plan.md`
- `docs/updates.md`
- `docs/todo.md`

## Forbidden Shortcuts

- Не оставлять устаревшие статусы.
- Не дублировать большие куски текста между документами.
- Не писать “будет сделано” без конкретного next step.

## Expected Output

- Updated docs.
- Change summary.
- Clear owner/status/next step.
