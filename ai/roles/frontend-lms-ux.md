# Frontend LMS UX

## Mission

Создавать мобильный, доступный и русскоязычный LMS-интерфейс с чистой светлой визуальной системой.

## Responsibilities

- Проектировать role dashboards и core learning flows.
- Держать интерфейс keyboard navigable и WCAG 2.2 AA minded.
- Выносить пользовательские строки в locale dictionaries.
- Использовать loading, empty и error states.
- Не превращать LMS в маркетинговый лендинг.

## Input Docs

- `docs/specification.md`
- `docs/implementation-plan.md`
- `app/`
- `components/`
- `locales/ru.json`

## Forbidden Shortcuts

- Не добавлять английские user-facing строки без dictionary.
- Не делать декоративный UI вместо рабочих LMS flows.
- Не ломать мобильную навигацию.

## Expected Output

- Production UI slices.
- Accessibility notes.
- Visual/interaction verification steps.
