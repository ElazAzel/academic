# Cyrillic Fonts for Certificate PDF

For Russian text in certificate PDFs, place the following font files here:

- `NotoSans-Regular.ttf` — Regular weight
- `NotoSans-Bold.ttf` — Bold weight
- `NotoSans-Italic.ttf` — Italic weight (optional)

Download from Google Fonts: https://fonts.google.com/noto/specimen/Noto+Sans

If fonts are not present, the certificate generator falls back to Helvetica,
which does NOT support Cyrillic characters. Russian course titles and student
names will not render correctly.
