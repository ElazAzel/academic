# Positioning: AI Strategic Academy

> Дата актуализации: 2026-06-15

---

## Что такое AI Strategic Academy

AI Strategic Academy — это **корпоративная AI-академия под ключ**. Закрытая платформа, которая обучает сотрудников практическому применению AI в рабочих задачах и показывает заказчику измеримый результат.

Платформа не является:

- публичным маркетплейсом курсов;
- универсальным LMS-шаблоном;
- библиотекой открытого контента;
- конструктором курсов общего назначения.

AI Strategic Academy — это операционная платформа для одного бизнеса: управление обучением, сопровождение кураторами, фиксация результатов и отчётность для заказчика.

---

## Кому продаётся

| Сегмент | Пример |
|---------|--------|
| Бизнес (HR / L&D) | Компания на 500+ сотрудников, внедряющая AI-культуру |
| Государственные организации | Госорган, цифровизирующий рабочие процессы |
| Университеты / академии | Вуз, запускающий программу AI-грамотности для студентов и преподавателей |
| Корпоративные академии | Внутренняя академия крупной компании, которой нужен готовый AI-курс |

## Что продаётся первым

Первый продукт: **AI Productivity Bootcamp** — 4-недельная программа интенсивного практического обучения.

| Параметр | Значение |
|----------|----------|
| Длительность | 4 недели |
| Группа | от 30 сотрудников |
| Цена | 160 000–260 000 KZT / чел. |
| Минимальный чек | от 5 000 000 KZT за поток |
| Формат | Обучение под ключ через закрытую LMS |
| Кураторы | Со стороны AI Strategic Academy |
| Заказчик | HR / L&D / руководитель проекта |
| Доступ | Логины выдаются участникам, публичной регистрации нет |
| Отчётность | Еженедельный отчёт + финальный отчёт |
| Итог | Сертификат + база знаний/артефактов + финальная автоматизация рабочей задачи |

---

## Чем отличается от обычной LMS

| Обычная LMS | AI Strategic Academy |
|-------------|---------------------|
| Предоставляет уроки | Предоставляет **управляемый результат** |
| Студент учится сам | Студента **сопровождает куратор** |
| Прогресс — галочка в списке | Прогресс — **PRODUCTIVITY SCORE** |
| Задания сдал — забыл | Задания = **артефакты автоматизации** |
| Сертификат — PDF | Сертификат = **доказательство навыка** |
| Отчёт — таблица | Отчёт = **бизнес-документ для заказчика** |
| Риски — скрыты | Риски = **основа для действий куратора** |
| Успех = прошёл курс | Успех = **рабочая задача автоматизирована** |

Главная ценность:

> **Не LMS сама по себе, а управляемый корпоративный AI-результат.**

## Пользователи и роли

| Роль | Описание |
|------|----------|
| **Admin** | Полное управление: пользователи, курсы, потоки, настройки, отчёты, аудит |
| **Super Curator** | Распределение кураторов, контроль SLA, глобальные риски, сводные отчёты |
| **Curator** | Сопровождение студентов: вопросы, проверка заданий, риски, push-уведомления |
| **Instructor** | Учебный контент: курсы, модули, уроки, тесты, задания |
| **Student** | Прохождение обучения: уроки, тесты, задания, финальная работа, сертификат |
| **Customer Observer** | Read-only доступ заказчика к своим потокам: прогресс, риски, отчёты, сертификаты |

## Главный коммерческий сценарий

30 сотрудников компании проходят AI Productivity Bootcamp за 4 недели:

1. Компания (HR / руководитель) заключает договор.
2. AI Strategic Academy создаёт поток, выдаёт логины.
3. Студенты проходят обучение: смотрят уроки, сдают тесты, выполняют задания.
4. Кураторы сопровождают: отвечают на вопросы, проверяют задания, выявляют риски.
5. Каждый студент сдаёт финальную работу — AI-автоматизацию своей рабочей задачи.
6. Заказчик получает еженедельные отчёты + финальный отчёт.
7. Завершившие получают сертификаты.
8. Заказчик видит: кто чему научился, какие задачи автоматизированы, какой ROI.

Ключевой outcome:

> **Каждый участник сдаёт AI-автоматизацию одной реальной рабочей задачи.**
