## 1. Weekly Cohort Report

**Назначение:** еженедельный отчёт для заказчика (HR / L&D / руководитель проекта).

### Структура

```yaml
client: "[Название компании]"
project: "[Название проекта]"
cohort: "[Название потока]"
course: "AI Productivity Bootcamp"
week: "1/4"
period: "[Дата начала] – [Дата окончания]"
```

### Секции

#### 1.1. Executive Summary (1 абзац)

Краткий итог недели: что пройдено, какие успехи, какие риски.

## 2. Final Cohort Report

**Назначение:** итоговый отчёт по окончании программы.

### Структура

```yaml
client: "[Название компании]"
project: "[Название проекта]"
cohort: "[Название потока]"
course: "AI Productivity Bootcamp"
period: "[Дата начала] – [Дата окончания]"
generated: "[Дата генерации]"
```

### Секции

#### 2.1. Executive Summary (2–3 абзаца)

- Итог: сколько завершили, сколько сдали финальную работу
