# Success Metrics: AI Strategic Academy

> Дата актуализации: 2026-06-15

---

## Product Success — как понять, что платформа работает как корпоративный AI-продукт

### Критические метрики (gate)

Эти метрики — обязательный минимум. Без них продукт не показывает ценность.

| # | Метрика | Цель | Как измеряется |
|---|---------|:----:|----------------|
| 1 | Сотрудники завершили курс | ≥80% | CourseProgress.completed / Enrollment.active |
| 2 | Сдали финальную работу | ≥70% | Final submission count / Enrollment.total |
| 3 | Loyalty / Satisfaction | ≥4.0 / 5 | Post-course survey (опрос завершения) |
| 4 | Каждый завершивший получил сертификат | 100% | Certificate.issued ≥ CourseProgress.completed |
| 5 | Заказчик получил еженедельные отчёты | 100% | Report.generated per week |
| 6 | Заказчик получил финальный отчёт | 100% | Report.generated (final type) |
| 7 | Есть список автоматизированных задач | ✅ | Export final project artifacts |
| 8 | Есть AI Productivity Score по участнику | ≥60 avg | ProductScore.calculated per participant |
| 9 | Есть сводка рисков и действий кураторов | ✅ | Curator action log |

## Как метрики привязаны к ролям

| Метрика | Admin | Curator | Customer Observer |
|---------|-------|---------|-------------------|
| % завершения | ✅ | ✅ (свои) | ✅ (свой поток) |
| % финальных работ | ✅ | ✅ (свои) | ✅ (свой поток) |
| AI Productivity Score | ✅ | ✅ (свои) | ✅ (свой поток) |
| Satisfaction / NPS | ✅ | агрегировано | агрегировано |
| Риски | ✅ | ✅ (свои) | ✅ (свой поток) |
| Отчёты | ✅ | — | ✅ (свой поток) |
| Сертификаты | ✅ | ✅ (свои) | ✅ (свой поток) |
| Артефакты | ✅ | ✅ (свои) | ✅ (свой поток) |
| Время ответа куратора | ✅ | ✅ (своё) | — |
| Uptime / performance | ✅ | — | — |

Customer Observer **не видит**:
- чужие потоки;
- персональные данные других заказчиков;
- внутренние метрики платформы;
- сырые технические данные.
